Clases de Job SAPSAP Job Classes

A Clase A — Alta PrioridadClass A — High Priority

Jobs criticos que deben ejecutarse en su ventana programada. Tienen prioridad sobre clases B y C en la asignacion de work processes. Critical jobs that must run in their scheduled window. They have priority over class B and C in work process allocation.

B Clase B — Prioridad MediaClass B — Medium Priority

Jobs de negocio regulares. Se ejecutan cuando hay work processes disponibles despues de los jobs clase A. Regular business jobs. Run when work processes are available after class A jobs.

C Clase C — Baja PrioridadClass C — Low Priority

Jobs de baja prioridad (reportes, housekeeping). Se ejecutan con los work processes sobrantes. Low priority jobs (reports, housekeeping). Run with remaining work processes.

Mejores PracticasBest Practices

💡
Monitoreo de Jobs Clase A: Los jobs de clase A son criticos para el negocio. Configure alertas para recibir notificaciones inmediatas si un job clase A falla o excede su duracion esperada. Class A Job Monitoring: Class A jobs are business-critical. Configure alerts to receive immediate notifications if a class A job fails or exceeds its expected duration.
⚠
Jobs fallidos recurrentes: Si un job falla repetidamente, revise el campo details para extraer el tipo de dump ABAP, el programa y la linea de error. Consulte las SAP Notes recomendadas. Recurring failed jobs: If a job fails repeatedly, review the details field to extract the ABAP dump type, program, and error line. Check the recommended SAP Notes.
