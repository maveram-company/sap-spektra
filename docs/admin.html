Esta pagina requiere el rol Admin. Solo los administradores de la organizacion pueden ver y gestionar los sistemas registrados desde este panel. This page requires the Admin role. Only organization administrators can view and manage registered systems from this panel.

Encabezado y Boton de ConexionHeader & Connect Button

En la parte superior de la pagina veras el titulo "Administracion" junto con la descripcion "Gestion de sistemas y configuracion". A la derecha del encabezado se encuentra el boton principal de accion: At the top of the page you will see the title "Administracion" alongside the description "System management and configuration". On the right side of the header you will find the main action button:

Gestion de sistemas y configuracion System management and configuration
+ Conectar SistemaConnect System

Al hacer clic en "Conectar Sistema", seras redirigido al asistente de conexion donde podras registrar un nuevo sistema SAP paso a paso. Consulta la guia de Conectar Sistema para mas detalles. Clicking "Connect System" takes you to the connection wizard where you can register a new SAP system step by step. See the Connect System guide for details.

Tabla de SistemasSystems Table

El cuerpo principal de la pagina muestra una tabla con todos los sistemas SAP registrados en tu organizacion. Cada fila es clickeable — al hacer clic en cualquier parte de la fila, navegaras a la pagina de detalle de ese sistema. The main body of the page displays a table with all SAP systems registered in your organization. Each row is clickable — clicking anywhere on the row navigates you to that system's detail page.

Columna: Base de DatosColumn: Database

Muestra el motor de base de datos subyacente del sistema. Los valores posibles incluyen: Shows the underlying database engine for the system. Possible values include:

Columna: AmbienteColumn: Environment

Un badge de color que identifica el entorno de despliegue del sistema. El color cambia segun el tipo de ambiente: A colored badge identifying the system's deployment environment. The color changes based on the environment type:

PRD QAS DEV
  • PRD (Produccion) — Badge rojo. Sistemas en produccion activa. PRD (Production) — Red badge. Active production systems.
  • QAS (Quality Assurance) — Badge amarillo. Sistemas de pruebas y calidad. QAS (Quality Assurance) — Yellow badge. Testing and quality systems.
  • DEV (Desarrollo) — Badge azul/cian. Sistemas de desarrollo. DEV (Development) — Blue/cyan badge. Development systems.

Trial Production

Columna: AccionesColumn: Actions

La ultima columna contiene 3 botones de icono para acciones rapidas sobre cada sistema: The last column contains 3 icon buttons for quick actions on each system:

Health CheckHealth Check (↻)
Al hacer clic, el icono comienza a girar (animacion de spin) indicando que se esta ejecutando una verificacion de salud del sistema en tiempo real. Cuando el chequeo termina, el icono deja de girar y la puntuacion de Health en la tabla se actualiza automaticamente con el nuevo valor. When clicked, the icon starts spinning (spin animation) indicating that a real-time health check is being performed on the system. When the check finishes, the icon stops spinning and the Health score in the table updates automatically with the new value.
EditarEdit (✎)
Navega a la pagina de detalle del sistema, donde puedes ver toda la informacion del sistema, modificar sus datos de configuracion, revisar metricas historicas y gestionar sus conectores. Navigates to the system detail page, where you can view all the system's information, modify its configuration data, review historical metrics, and manage its connectors.
EliminarDelete (🗑)
Requiere doble clic como medida de seguridad. Al hacer el primer clic, el boton cambia a color rojo y muestra un texto de confirmacion (por ejemplo, "Confirmar"). Solo al hacer el segundo clic dentro de unos segundos se procedera a eliminar el sistema. Si no confirmas, el boton vuelve a su estado normal. Requires a double click as a safety measure. On the first click, the button turns red and shows a confirmation text (e.g., "Confirm"). Only clicking a second time within a few seconds will proceed with deleting the system. If you don't confirm, the button returns to normal.

Navegacion desde las FilasRow Navigation

Todas las filas de la tabla son clickeables. Al hacer clic en cualquier parte de una fila (excepto en los botones de accion), seras redirigido a la pagina de detalle del sistema, donde encontraras informacion completa incluyendo metricas en tiempo real, historicos de rendimiento, conectores asociados y opciones de configuracion avanzada. All table rows are clickable. Clicking anywhere on a row (except on the action buttons) will redirect you to the system detail page, where you'll find complete information including real-time metrics, performance history, associated connectors, and advanced configuration options.
