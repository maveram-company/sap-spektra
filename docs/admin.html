Metodos de ConexionConnection Methods

Spektra soporta multiples metodos de conexion a sistemas SAP. Para agregar un nuevo sistema, use el asistente de conexion. Spektra supports multiple connection methods to SAP systems. To add a new system, use the connection wizard.

☁️
SAP Cloud Connector
RecomendadoRecommended

Conexion segura via SAP BTP Cloud Connector. Ideal para sistemas en la nube y on-premise con BTP.Secure connection via SAP BTP Cloud Connector. Ideal for cloud and on-premise systems with BTP.

📡
Spektra Agent
On-Premise

Agente ligero instalado directamente en el servidor SAP. Comunicacion cifrada via WebSocket.Lightweight agent installed directly on the SAP server. Encrypted communication via WebSocket.

🔌
RFC/BAPI
Legacy

Conexion directa via RFC para sistemas legacy. Requiere apertura de puertos y usuario de servicio.Direct RFC connection for legacy systems. Requires port opening and service user.

🌐
API Gateway
REST

Conexion via SAP API Gateway o SAP Integration Suite para APIs OData y REST.Connection via SAP API Gateway or SAP Integration Suite for OData and REST APIs.
