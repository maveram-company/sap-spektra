Vista General Overview

El Dashboard es la pagina principal de SAP Spektra. Proporciona una vista de alto nivel del estado completo del landscape SAP, permitiendo a los operadores identificar rapidamente problemas, actuar sobre alertas y monitorear la salud general de todos los sistemas. The Dashboard is the main page of SAP Spektra. It provides a high-level view of the complete SAP landscape status, allowing operators to quickly identify issues, act on alerts and monitor the overall health of all systems.

Cache y Rendimiento Cache and Performance

💡
Cache Redis: El endpoint GET /dashboard utiliza cache Redis con TTL de 30 segundos. Esto permite que multiples usuarios vean el dashboard sin saturar la base de datos con consultas agregadas pesadas. Redis Cache: The GET /dashboard endpoint uses Redis cache with a 30-second TTL. This allows multiple users to view the dashboard without overloading the database with heavy aggregation queries.
