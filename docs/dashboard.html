DashboardDashboard

Vista general del estado de tu landscape SAP en tiempo real Real-time overview of your SAP landscape health

Barra de EncabezadoHeader Bar

Al abrir el Dashboard, la parte superior de la pagina muestra un saludo personalizado con el formato "Hola, [tu nombre de usuario]". Debajo del saludo aparece el nombre de tu organizacion y la fecha actual. When you open the Dashboard, the top of the page displays a personalized greeting in the format "Hola, [your username]". Below the greeting, you will see your organization name and the current date.

A la derecha del encabezado hay un boton de refrescar que permite actualizar manualmente todos los datos de la pagina sin necesidad de recargar el navegador. On the right side of the header, there is a refresh button that lets you manually update all page data without reloading the browser.

Tarjetas KPIKPI Cards

Justo debajo del encabezado se presenta una grilla de 4 tarjetas KPI que resumen el estado general de tu landscape. Cada tarjeta incluye un icono circular con gradiente, un numero grande con efecto de brillo (glow), una etiqueta descriptiva y un indicador de cambio porcentual. Just below the header, a grid of 4 KPI cards summarizes the overall state of your landscape. Each card includes a gradient circle icon, a large number with a glow effect, a descriptive label, and a percentage change indicator.

Cada tarjeta KPI muestra un indicador de cambio porcentual respecto al periodo anterior (por ejemplo, +12% o -5%). Una flecha verde hacia arriba indica mejora, y una roja hacia abajo indica degradacion. Each KPI card shows a percentage change indicator compared to the previous period (for example, +12% or -5%). A green upward arrow indicates improvement, and a red downward arrow indicates degradation.

Seccion Landscape SAPSAP Landscape Section

Debajo de las tarjetas KPI, veras el encabezado "Landscape SAP" acompanado de una pastilla (pill) que muestra el numero total de sistemas registrados. Por ejemplo: 12 sistemas. Below the KPI cards, you will see the heading "Landscape SAP" accompanied by a pill badge showing the total number of registered systems. For example: 12 systems.

Grilla de Tarjetas de SistemaSystem Cards Grid

Los sistemas se muestran en una grilla responsive de 1 a 4 columnas. Cada tarjeta de sistema contiene la siguiente informacion: Systems are displayed in a responsive grid of 1 to 4 columns. Each system card contains the following information:

Encabezado de la tarjetaCard Header
  • SID — El identificador del sistema (por ejemplo, S4P, ECP) mostrado con texto en gradiente que lo hace visualmente destacado. SID — The system identifier (for example, S4P, ECP) displayed with gradient text that makes it visually prominent.
  • Badge de estado — Un indicador de color junto al SID que refleja el estado actual del sistema: Healthy Warning Degraded Critical Status badge — A colored indicator next to the SID reflecting the current system status: Healthy Warning Degraded Critical
  • Descripcion — Un texto corto que describe el sistema, junto a un badge de modo: Production o Trial. Description — A short text describing the system, alongside a mode badge: Production or Trial.
Health Gauge (Indicador de Salud)Health Gauge

En el centro de cada tarjeta de sistema se muestra un gauge circular SVG que indica la puntuacion de salud del sistema en una escala de 0 a 100. El color del arco del gauge cambia segun la puntuacion: At the center of each system card, a circular SVG gauge displays the system's health score on a scale of 0 to 100. The gauge arc color changes based on the score:

  • Verde — Puntuacion alta (sistema saludable, tipicamente 80–100) Green — High score (healthy system, typically 80–100)
  • Ambar — Puntuacion media (advertencia, tipicamente 50–79) Amber — Medium score (warning, typically 50–79)
  • Rojo — Puntuacion baja (critico, tipicamente 0–49) Red — Low score (critical, typically 0–49)

El numero de la puntuacion aparece en grande dentro del circulo con un efecto de brillo. The score number appears large inside the circle with a glow effect.

Acciones RapidasQuick Actions

Si existen aprobaciones pendientes (el KPI de aprobaciones es mayor a 0), en la parte inferior del Dashboard aparece una tarjeta de alerta de color ambar que informa al usuario que hay solicitudes esperando revision. If there are pending approvals (the approvals KPI is greater than 0), an amber-colored alert card appears at the bottom of the Dashboard informing the user that there are requests awaiting review.

⚠️
Esta tarjeta incluye un boton "Ver Aprobaciones" que te lleva directamente a la pagina de Aprobaciones para que puedas revisar y actuar sobre las solicitudes pendientes. This card includes a "Ver Aprobaciones" (View Approvals) button that takes you directly to the Approvals page so you can review and act on pending requests.

Si no hay aprobaciones pendientes, esta seccion no se muestra — el Dashboard mantiene un diseno limpio sin elementos innecesarios. If there are no pending approvals, this section is not displayed — the Dashboard keeps a clean layout without unnecessary elements.

Resumen VisualVisual Summary

El Dashboard esta organizado de arriba hacia abajo en el siguiente orden: The Dashboard is organized from top to bottom in the following order:

Encabezado — Saludo personalizado, nombre de organizacion, fecha y boton de refrescar. Header — Personalized greeting, organization name, date, and refresh button.
Tarjetas KPI — 4 tarjetas en grilla con metricas clave: sistemas activos, saludables, breaches y aprobaciones. KPI Cards — 4 cards in a grid with key metrics: active systems, healthy systems, breaches, and approvals.
Landscape SAP — Encabezado con conteo total de sistemas y grilla de tarjetas de sistema. SAP Landscape — Heading with total system count and system card grid.
Tarjetas de Sistema — Grilla de 1-4 columnas donde cada tarjeta muestra SID, estado, health gauge y metricas. System Cards — 1-4 column grid where each card shows SID, status, health gauge, and metrics.
Acciones Rapidas — Alerta ambar con boton para ver aprobaciones pendientes (solo visible si hay pendientes). Quick Actions — Amber alert with button to view pending approvals (only visible when there are pending items).
