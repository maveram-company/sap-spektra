Camino de Transporte del LandscapeLandscape Transport Path

El camino de transporte define la ruta que sigue cada orden desde desarrollo hasta produccion. Cada transporte debe ser liberado en el sistema origen e importado en los sistemas destino secuencialmente. The transport path defines the route each request follows from development to production. Each transport must be released in the source system and imported into target systems sequentially.

Estados de TransporteTransport Statuses

  • modifiable — Orden en desarrollo, no liberada aunRequest under development, not yet released
  • released — Liberada, pendiente de importacion en destinoReleased, pending import in target system
  • imported — Importada exitosamente en el sistema destinoSuccessfully imported in target system
  • error — Error durante la importacionError during import

Codigos de Retorno (RC)Return Codes (RC)

  • RC 0 — Exito total, sin advertenciasComplete success, no warnings
  • RC 4 — Advertencias menores (generalmente aceptable)Minor warnings (generally acceptable)
  • RC 8 — Errores significativos, requiere revisionSignificant errors, requires review
  • RC 12+ — Error critico, importacion fallidaCritical error, import failed

Mejores PracticasBest Practices

💡
Importacion en horario controlado: Programe las importaciones a produccion durante ventanas de mantenimiento. Los transportes con cambios DDIC pueden bloquear tablas temporalmente. Controlled import schedule: Schedule production imports during maintenance windows. Transports with DDIC changes may temporarily lock tables.
⚠
Return Code 4: Generalmente son advertencias de activacion que no afectan la funcionalidad. Sin embargo, se recomienda revisarlos antes de continuar al siguiente sistema del landscape. Return Code 4: These are usually activation warnings that don't affect functionality. However, it's recommended to review them before continuing to the next landscape system.
⚠
Return Code 8+: Indica errores reales. No importe este transporte al siguiente sistema hasta resolver el error. Verifique dependencias, includes faltantes, y objetos bloqueados. Return Code 8+: Indicates real errors. Do not import this transport to the next system until the error is resolved. Check dependencies, missing includes, and locked objects.
