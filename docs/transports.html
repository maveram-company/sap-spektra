Formato del ID de TransporteTransport ID Format

Los transportes SAP siguen un formato estandar SIDKNNNNNN donde: SAP transports follow a standard format SIDKNNNNNN where:

ComponenteComponent LongitudLength DescripcionDescription EjemploExample
SID 3 System Identifier del sistema origen donde se creo el transporteSystem Identifier of the source system where the transport was created DEV
K 1 Prefijo fijo que indica "transport request" (K = Korrektur/Correction)Fixed prefix indicating "transport request" (K = Korrektur/Correction) K
NNNNNN 6 Numero secuencial unico dentro del sistema origenUnique sequential number within the source system 900123 
// Transport ID validation
const TRANSPORT_REGEX = /^[A-Z]{3}K\d{6}$/;

// Valid examples
"DEVK900123"  // Transport from DEV system, number 900123
"PRDK000001"  // Transport from PRD system, number 000001
"QASK800456"  // Transport from QAS system, number 800456

// Invalid examples
"DEV900123"   // Missing 'K' prefix
"DEVK90012"   // Number too short (5 digits instead of 6)
"devk900123"  // Lowercase SID not allowed

Modelo de DatosData Model

interface SapTransport {
  id:              string;          // Internal UUID
  transportId:     string;          // SAP transport ID (e.g., "DEVK900123")
  description:     string;          // Transport description
  owner:           string;          // SAP user who created it
  sourceSystem:    string;          // Source SID (e.g., "DEV")
  targetSystem:    string;          // Current target SID (e.g., "QAS")
  status:          TransportStatus; // released | imported | error | modifiable
  returnCode:      number | null;   // 0, 4, 8, 12, 16
  type:            TransportType;   // WORKBENCH | CUSTOMIZING
  category:        string;          // K (Correction) | T (Task)
  releasedAt:      ISO8601 | null;  // When released
  importedAt:      ISO8601 | null;  // When imported to target
  objects:         TransportObject[];// List of transport objects
  importLog:       string | null;   // Import log text
  organizationId:  string;
  createdAt:       ISO8601;
  updatedAt:       ISO8601;
}
