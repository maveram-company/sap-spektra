Integracion con Amazon BedrockAmazon Bedrock Integration

SAP Spektra utiliza Amazon Bedrock como servicio de inteligencia artificial, especificamente el modelo Anthropic Claude (claude-3-sonnet / claude-3-haiku). Bedrock proporciona acceso gestionado a modelos de lenguaje grandes (LLMs) sin necesidad de gestionar infraestructura de ML. SAP Spektra uses Amazon Bedrock as its AI service, specifically the Anthropic Claude model (claude-3-sonnet / claude-3-haiku). Bedrock provides managed access to large language models (LLMs) without needing to manage ML infrastructure.

ℹ
Privacidad: Las consultas a Bedrock son procesadas dentro de la region AWS configurada. Los datos del cliente no se utilizan para entrenar modelos. Cada peticion incluye solo el contexto minimo necesario para la respuesta. Privacy: Bedrock queries are processed within the configured AWS region. Customer data is not used to train models. Each request includes only the minimum context necessary for the response.

Mecanismo de Keyword MatchingKeyword Matching Mechanism

El motor de IA utiliza un sistema de keyword matching para identificar la intencion del usuario y seleccionar el caso de uso apropiado. Las keywords se evaluan en espanol e ingles. The AI engine uses a keyword matching system to identify user intent and select the appropriate use case. Keywords are evaluated in both Spanish and English. 

// Keyword matching configuration
const USE_CASE_KEYWORDS = {
  CPU_INCIDENT:    ['cpu', 'alto', 'high', 'procesador', 'processor', 'work process'],
  DISK_PREDICTIVE: ['disco', 'disk', 'espacio', 'space', 'storage', 'lleno', 'full'],
  SAFETY_GATE:     ['seguro', 'safe', 'gate', 'riesgo', 'risk', 'validar', 'validate'],
  EXEC_DIGEST:     ['resumen', 'summary', 'digest', 'ejecutivo', 'executive'],
  ADAPTATION:      ['adaptar', 'adapt', 'optimizar', 'optimize', 'tuning', 'mejorar'],
  GENERAL_HEALTH:  ['salud', 'health', 'estado', 'status', 'como esta', 'how is'],
  RESTART_RISK:    ['restart', 'reinicio', 'reboot', 'apagar', 'shutdown', 'parar']
};

// Matching algorithm: score-based with threshold
function matchUseCase(userMessage: string): UseCase | null {
  const normalized = userMessage.toLowerCase();
  let bestMatch = null;
  let bestScore = 0;

  for (const [useCase, keywords] of Object.entries(USE_CASE_KEYWORDS)) {
    const score = keywords.filter(kw => normalized.includes(kw)).length;
    if (score > bestScore) {
      bestScore = score;
      bestMatch = useCase;
    }
  }

  return bestScore >= 1 ? bestMatch : null; // Fallback to general chat
}
