Analisis IA AI Analysis

Amazon Bedrock (Claude) — 7 casos de uso use cases

Requisito del plan Plan requirement

La funcionalidad de Analisis IA esta disponible unicamente en los planes que incluyen el modulo ai_analysis. Si tu plan actual no lo incluye, veras un banner en el centro de la pantalla invitandote a actualizar tu suscripcion. The AI Analysis feature is available only on plans that include the ai_analysis module. If your current plan does not include it, you will see a banner in the center of the screen inviting you to upgrade your subscription.

🔒 Funcionalidad premium Premium feature

Si ves este banner, tu plan no incluye Analisis IA con Bedrock. Contacta a tu administrador o actualiza tu plan para desbloquear esta seccion. If you see this banner, your plan does not include AI Analysis with Bedrock. Contact your admin or upgrade your plan to unlock this section.

Tarjetas de casos de uso Use case cards

La parte superior de la pagina muestra una cuadricula de 7 tarjetas, distribuidas en 1 a 3 columnas segun el ancho de pantalla. Cada tarjeta representa un caso de uso de IA especializado para monitoreo SAP. The top of the page displays a grid of 7 cards, laid out in 1 to 3 columns depending on screen width. Each card represents a specialized AI use case for SAP monitoring.

Estructura de cada tarjeta Card structure

  • Borde izquierdo de color — identifica la categoria de la tarjeta (rojo para peligro, amarillo para advertencia, cian para primario, violeta para acento, verde para exito). Color-coded left border — identifies the card category (red for danger, yellow for warning, cyan for primary, violet for accent, green for success).
  • ID coloreado — etiqueta corta como UC1, UC2, etc., en el color correspondiente. Colored ID label — short tag like UC1, UC2, etc., in the matching color.
  • Nombre — titulo descriptivo del caso de uso. Name — descriptive title of the use case.
  • Descripcion — breve explicacion de lo que hace el caso de uso. Description — brief explanation of what the use case does.
  • Click — al hacer clic en cualquier tarjeta, se envia la consulta predefinida del caso de uso directamente al chat de IA (la seccion inferior). Click — clicking any card sends the pre-defined use case query directly to the AI chat (lower section).

Los 7 casos de uso The 7 use cases

UC1 Analisis Incidentes Incident Analysis

Diagnostico en tiempo real de breaches. Analiza incidentes de CPU, memoria u otros recursos criticos. Real-time breach diagnosis. Analyzes CPU, memory or other critical resource incidents.

UC2 Prediccion Disco Disk Prediction

Forecast de crecimiento de disco con regresion lineal. Predice cuando un filesystem se quedara sin espacio. Disk growth forecast with linear regression. Predicts when a filesystem will run out of space.

UC3Safety Gate

Evaluacion de seguridad pre-ejecucion. Verifica si es seguro ejecutar un runbook en un sistema en este momento. Pre-execution safety assessment. Checks if it is safe to execute a runbook on a system right now.

UC4 Digest Ejecutivo Executive Digest

Resumen diario automatico generado a las 22:00 UTC. Panorama completo del estado del landscape. Automatic daily summary generated at 22:00 UTC. Full overview of landscape health.

UC5 Adaptacion Runbooks Adaptive Thresholds

Ajustes especificos por sistema. Adapta runbooks a las particularidades de cada entorno SAP (tipo de DB, version, etc.). System-specific tuning. Adapts runbooks to the specifics of each SAP environment (DB type, version, etc.).

UC6 Chatbot (Estado General) Chatbot (System Status)

Interfaz conversacional de autoservicio. Consulta el estado general de todos los sistemas en lenguaje natural. Self-service conversational interface. Query the overall status of all systems in natural language.

UC7 Evaluacion de Riesgo Risk Assessment

Evalua el riesgo de operaciones programadas. Analiza si un reinicio o cambio en un sistema es seguro en el horario actual. Evaluates risk of scheduled operations. Analyzes whether a restart or change on a system is safe at the current time.
