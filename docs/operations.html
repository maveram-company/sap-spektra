Ejecucion de Comandos via AgenteCommand Execution via Agent

Las operaciones se ejecutan enviando comandos al agente Spektra instalado en el servidor SAP. El agente ejecuta los comandos de forma segura y reporta resultados en tiempo real. Operations are executed by sending commands to the Spektra agent installed on the SAP server. The agent executes commands securely and reports results in real time. 

// Command sent to agent via heartbeat response

Modo de Ejecucion SimuladaSimulated Execution Mode

ℹ
Modo Demo: En modo simulado (EXECUTION_MODE=LOCAL_SIMULATED), los comandos no se envian al agente. En su lugar, el backend simula la ejecucion con tiempos de espera realistas y resultados pre-configurados. Esto permite demostrar y probar el flujo de operaciones sin impactar sistemas reales. Demo Mode: In simulated mode (EXECUTION_MODE=LOCAL_SIMULATED), commands are not sent to the agent. Instead, the backend simulates execution with realistic wait times and pre-configured results. This allows demonstrating and testing the operations flow without impacting real systems.

Programacion con CronCron Scheduling

Las operaciones pueden programarse para ejecucion recurrente usando expresiones cron estandar. El scheduler de Spektra evalua las expresiones cada minuto y crea instancias de operacion automaticamente. Operations can be scheduled for recurring execution using standard cron expressions. Spektra's scheduler evaluates expressions every minute and creates operation instances automatically.
