Tipos de OperacionOperation Types

💾
BACKUP

Respaldos completos o incrementales de bases de datos y sistemas de archivos SAP. Full or incremental backups of databases and SAP file systems.

🔄
RESTART

Reinicio de instancias SAP, servicios o componentes individuales. Restart of SAP instances, services, or individual components.

🔧
MAINTENANCE

Ventanas de mantenimiento planificadas: patches, upgrades, configuracion. Planned maintenance windows: patches, upgrades, configuration changes.

💥
DR_DRILL

Simulacros de recuperacion ante desastres y pruebas de failover. Disaster recovery drills and failover testing exercises.

🗡
HOUSEKEEPING

Limpieza de logs, archivos temporales, reorganizacion de tablas, purge de datos. Log cleanup, temp files, table reorganization, data purge tasks.

Estados y Nivel de RiesgoStatuses & Risk Level

EstadosStatuses

  • SCHEDULED — Programada, pendiente de ejecucionScheduled, pending execution
  • RUNNING — En ejecucion activaActively running
  • COMPLETED — Finalizada exitosamenteCompleted successfully
  • FAILED — Fallo durante la ejecucionFailed during execution
  • CANCELLED — Cancelada antes de ejecutarseCancelled before execution

Niveles de RiesgoRisk Levels

  • LOW
    Sin impacto en produccion (housekeeping, backups incrementales)No production impact (housekeeping, incremental backups)
  • MEDIUM
    Impacto menor posible (patches non-critical, config changes)Possible minor impact (non-critical patches, config changes)
  • HIGH
    Downtime potencial (reinicios, upgrades de kernel)Potential downtime (restarts, kernel upgrades)
  • CRITICAL
    Downtime garantizado o riesgo de datos (DR drills, full system restart)Guaranteed downtime or data risk (DR drills, full system restart)
