Referencia APIAPI Reference

GET /api/analytics/overview

Obtiene los KPIs agregados del landscape completo. Gets aggregated KPIs for the entire landscape. 

GET /api/analytics/overview

Response: {
  "totalSystems": 24,
  "avgHealth": 84.2,
  "totalAlerts7d": 47,
  "totalOperations30d": 156,
  "systemsByEnv": { "PRD": 10, "QAS": 6, "DEV": 5, "SBX": 3 }
}

GET /api/analytics/runbooks

Estadisticas detalladas de ejecucion de runbooks. Detailed runbook execution statistics. 

GET /api/analytics/runbooks?range=30d&sort=executions

Response: {
  "totalExecutions": 342,
  "successRate": 0.947,
  "avgDuration": 144,
  "runbooks": [
    { "name": "RB-HEALTH-CHECK", "executions": 89, "successRate": 1.0 },
    ...
  ]
}

GET /api/analytics/health-trends

Tendencias de salud de sistemas en la ventana de tiempo seleccionada. System health trends in the selected time window. 

GET /api/analytics/health-trends?range=14d&sid=S4P

Response: {
  "sid": "S4P",
  "range": "14d",
  "average": 95.8,
  "change": +0.4,
  "dataPoints": [
    { "date": "2026-02-27", "healthScore": 95 },
    { "date": "2026-02-28", "healthScore": 96 },
    ...
  ]
}
