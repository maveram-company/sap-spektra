Endpoint de Resumen de AnalyticsAnalytics Overview Endpoint

El endpoint /api/analytics/overview proporciona un resumen consolidado de todas las metricas del landscape en un solo llamado. Tendencias de Salud del SistemaSystem Health Trends

Las tendencias de salud proporcionan datos de series temporales para visualizar la evolucion de las metricas de cada sistema a lo largo del tiempo. Se soportan multiples granularidades. Health trends provide time-series data to visualize the evolution of each system's metrics over time. Multiple granularities are supported.

GranularidadGranularity IntervaloInterval Rango MaximoMaximum Range Uso RecomendadoRecommended Use

1d 1 dia1 day 90d Reportes semanales/mensualesWeekly/monthly reports
1w 1 semana1 week 365d Analisis de capacidad a largo plazoLong-term capacity analysis

Datos de Series TemporalesTime-Series Data

Spektra almacena datos de series temporales para las siguientes metricas. Cada metrica se almacena con su timestamp original y se agrega segun la granularidad solicitada. Spektra stores time-series data for the following metrics. Each metric is stored with its original timestamp and aggregated according to the requested granularity.

Metricas de InfraestructuraInfrastructure Metrics

  • CPU: Utilizacion total, por core, y por proceso SAPCPU: Total utilization, per core, and per SAP process
  • Memoria: Uso fisico, swap, buffer/cache HANAMemory: Physical usage, swap, HANA buffer/cache
  • Disco: Espacio usado/libre, I/O read/write, latencia I/ODisk: Used/free space, I/O read/write, I/O latency
  • Red: Bytes in/out, paquetes, erroresNetwork: Bytes in/out, packets, errors

Metricas de Aplicacion SAPSAP Application Metrics

  • Work Processes: Ocupados/libres por tipo (DIA, BTC, UPD)Work Processes: Busy/free by type (DIA, BTC, UPD)
  • Response Time: Tiempo de respuesta del dialogResponse Time: Dialog response time
  • Users: Usuarios conectados activosUsers: Active connected users
  • HANA: Memory allocation, column store size, delta mergesHANA: Memory allocation, column store size, delta merges

Agregacion del DashboardDashboard Aggregation

El dashboard principal consume datos agregados de multiples fuentes para presentar una vista unificada del landscape. Las agregaciones se calculan en tiempo real o se cachean por periodos cortos (30-60 segundos) para optimizar rendimiento. The main dashboard consumes aggregated data from multiple sources to present a unified landscape view. Aggregations are calculated in real time or cached for short periods (30-60 seconds) to optimize performance.
