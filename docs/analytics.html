Analytics

Metricas de ejecucion de runbooks Runbook execution metrics

Filtro de sistema System filter

En la parte superior de la pagina encontraras un dropdown de filtro que permite seleccionar un sistema especifico o ver los datos de todos los sistemas agregados. Por defecto esta seleccionado "Todos los sistemas". At the top of the page you will find a filter dropdown that lets you select a specific system or view aggregated data from all systems. By default "Todos los sistemas" (All systems) is selected.

Requisito del plan Plan requirement

Analytics requiere que tu plan incluya la funcionalidad analytics. Si no esta incluida, veras un banner de upgrade en lugar de las metricas y graficas. Analytics requires your plan to include the analytics feature. If it is not included, you will see an upgrade banner instead of metrics and charts.

🔒 Funcionalidad premium Premium feature

Si ves este banner, tu plan no incluye Analytics de Runbooks. Contacta a tu administrador o actualiza tu plan. If you see this banner, your plan does not include Runbook Analytics. Contact your admin or upgrade your plan.

Columnas de la tabla Table columns

  • Runbook — identificador del runbook en formato code (fuente monoespaciada con fondo gris). Runbook — runbook identifier in code format (monospaced font with grey background).
  • Nombre — nombre descriptivo del runbook en texto con peso medio. Name — descriptive runbook name in medium-weight text.
  • Ejecuciones — cantidad total de veces que se ha ejecutado este runbook. Executions — total number of times this runbook has been executed.
  • Tasa de Exito — porcentaje de ejecuciones exitosas, visualizado con una barra de progreso horizontal y el porcentaje numerico a la derecha. El color de la barra cambia: verde si > 95%, amarillo si > 90%, rojo si ≤ 90%. Success Rate — percentage of successful executions, shown with a horizontal progress bar and the numeric percentage on the right. Bar color changes: green if > 95%, yellow if > 90%, red if ≤ 90%.
