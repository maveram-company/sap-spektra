Docker Compose levanta PostgreSQL 16 y Redis 7 como contenedores. Los datos persisten en volumenes locales. Docker Compose starts PostgreSQL 16 and Redis 7 as containers. Data persists in local volumes.

Cambio de Modos Mode Switching

El modo operacional del frontend se controla programaticamente. Por defecto opera en modo REAL cuando el backend esta accesible. The frontend operational mode is controlled programmatically. By default it operates in REAL mode when the backend is reachable. 

// En codigo del frontend:
import { setDataServiceMode } from './services/dataService';

setDataServiceMode('REAL');       // Datos del backend
setDataServiceMode('MOCK');       // Datos simulados locales
setDataServiceMode('FALLBACK');   // Intenta real, cae a mock
setDataServiceMode('RESTRICTED'); // Restriccion para cloud-managed
💡
Si config.features.demoMode esta en true, el sistema opera automaticamente en modo MOCK para compatibilidad retroactiva. If config.features.demoMode is true, the system automatically operates in MOCK mode for backward compatibility.
