10 activosactive

Referencia APIAPI Reference

GET /api/connectors

Lista todos los conectores con estado y ultima actividad. Lists all connectors with status and last activity. 

GET /api/connectors?status=connected

Response: {
  "data": [
    {
      "sid": "S4P",
      "method": "spektra_agent",
      "status": "connected",
      "latencyMs": 12,
      "version": "2.4.1",
      "lastHeartbeat": "2026-03-12T10:00:02Z"
    }
  ]
}

GET /api/connectors/:sid/heartbeats

Obtiene el historial de heartbeats para un conector especifico. Gets the heartbeat history for a specific connector. 

GET /api/connectors/S4P/heartbeats?range=1h

Response: {
  "sid": "S4P",
  "availability": 100,
  "heartbeats": [
    { "timestamp": "...", "latencyMs": 12, "status": "ok" },
    ...
  ]
}
