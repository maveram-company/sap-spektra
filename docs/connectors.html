Gestione y supervise las conexiones entre sus sistemas SAP y la plataforma Spektra. Manage and monitor the connections between your SAP systems and the Spektra platform.

ConectoresConnectors

Conexiones de tus sistemas SAP con Spektra Connections from your SAP systems to Spektra

+ Conectar SistemaConnect System

Al abrir la pagina Conectores (Monitor > Conectores), lo primero que ve es el titulo junto con la descripcion "Conexiones de tus sistemas SAP con Spektra". El boton "Conectar Sistema" en la esquina superior derecha lo lleva al asistente para vincular un nuevo sistema. When you open the Connectors page (Monitor > Connectors), the first thing you see is the title alongside the description "Connections from your SAP systems to Spektra." The "Connect System" button in the upper-right corner takes you to the wizard for linking a new system.

Alerta de desconexionDisconnection alert

Cuando existe al menos una conexion en estado Desconectado, aparece automaticamente una tarjeta de alerta roja justo debajo de las tarjetas de resumen. Esta alerta le informa cuantas conexiones no responden y le orienta sobre que verificar: When at least one connection is in Disconnected status, a red alert card automatically appears just below the summary cards. This alert tells you how many connections are unresponsive and guides you on what to check:

⚠️
1 conexion sin respuesta 1 unresponsive connection
Verificar estado del agente o Cloud Connector. Revise la conectividad de red y que el servicio del agente Spektra este activo en el servidor correspondiente. Check the agent or Cloud Connector status. Review network connectivity and ensure the Spektra agent service is active on the corresponding server.
⚠️
Si el numero de desconexiones es mayor a uno, el mensaje se actualiza. Por ejemplo: "3 conexiones sin respuesta". La alerta desaparece automaticamente cuando todas las conexiones vuelven al estado Conectado o Degradado. If the disconnection count is greater than one, the message updates accordingly. For example: "3 unresponsive connections." The alert automatically disappears once all connections return to Connected or Degraded status.

Tabla de conexiones SAPSAP connections table

La seccion principal de la pagina es una tabla titulada "Conexiones SAP" que lista todas las conexiones configuradas. La tabla esta ordenada por prioridad: primero aparecen las conexiones desconectadas (rojo), luego las degradadas (amarillo) y finalmente las conectadas (verde). Esto asegura que los problemas mas urgentes sean visibles al instante. The main section of the page is a table titled "SAP Connections" listing all configured connections. The table is sorted by priority: disconnected connections (red) appear first, then degraded (yellow), and finally connected (green). This ensures the most urgent issues are visible at a glance.

Columnas de la tablaTable columns

ColumnaColumn DescripcionDescription
EstadoStatus Un punto de color seguido del nombre del estado.  Conectado: la conexion funciona correctamente.  Degradado: responde pero con rendimiento reducido.  Desconectado: no hay respuesta. A colored dot followed by the status name.  Connected: the connection is working correctly.  Degraded: responds but with reduced performance.  Disconnected: no response.
SID Identificador de tres letras del sistema SAP, mostrado como un badge con fondo cian. Ejemplo: P01. Three-letter SAP system identifier, shown as a badge with a cyan background. Example: P01.
SistemaSystem Nombre descriptivo del sistema (por ejemplo "SAP ERP PRD") con el tipo y ambiente debajo en texto pequeno (por ejemplo "ABAP • PRD"). Descriptive system name (e.g., "SAP ERP PRD") with the type and environment below in small text (e.g., "ABAP • PRD").
Metodo de ConexionConnection Method Badge que indica como se conecta el sistema a Spektra. Los metodos disponibles son: Badge indicating how the system connects to Spektra. Available methods are:
  • SAP Cloud Connector — Tunel seguro via SAP BTPSecure tunnel via SAP BTP
  • Spektra Agent — Agente instalado en el servidor SAPAgent installed on the SAP server
  • RFC/BAPI — Conexion directa por protocolo RFCDirect connection via RFC protocol
  • API Gateway — Conexion a traves de una pasarela APIConnection through an API gateway
LatenciaLatency Tiempo de respuesta del ultimo heartbeat. El valor se colorea segun umbrales: <100 ms (verde, optimo), 100–250 ms (amarillo, aceptable) y >250 ms (rojo, problematico). Si la conexion esta desconectada, se muestra un guion. Response time of the last heartbeat. The value is color-coded by threshold: <100 ms (green, optimal), 100–250 ms (yellow, acceptable), and >250 ms (red, problematic). If the connection is disconnected, a dash is shown.
Ultimo HeartbeatLast Heartbeat Tiempo transcurrido desde la ultima senal de vida recibida, en formato relativo (por ejemplo "hace 30s", "hace 2m", "hace 4h 23m"). Heartbeats antiguos se muestran en rojo. Time elapsed since the last received heartbeat, in relative format (e.g., "30s ago," "2m ago," "4h 23m ago"). Old heartbeats are shown in red.
Mensajes (24h)Messages (24h) Numero total de mensajes (datos, metricas, eventos) intercambiados entre el sistema y Spektra en las ultimas 24 horas. Un valor de 0 en una conexion activa puede indicar un problema de configuracion. Total number of messages (data, metrics, events) exchanged between the system and Spektra in the last 24 hours. A value of 0 on an active connection may indicate a configuration issue.
VersionVersion Version del agente o conector instalado. Mantenga todos los agentes actualizados a la ultima version estable para asegurar la compatibilidad y el acceso a las funcionalidades mas recientes. Version of the installed agent or connector. Keep all agents updated to the latest stable version to ensure compatibility and access to the newest features.

Orden de la tablaTable sort order

La tabla se ordena automaticamente por prioridad de estado para que los problemas mas criticos sean los primeros que usted ve: The table is automatically sorted by status priority so the most critical issues are the first you see:

DesconectadosDisconnected — Conexiones sin respuesta aparecen primero. Requieren atencion inmediata. Unresponsive connections appear first. They require immediate attention.
DegradadosDegraded — Conexiones con rendimiento reducido aparecen en segundo lugar. Deben ser investigadas. Connections with reduced performance appear second. They should be investigated.
ConectadosConnected — Conexiones saludables aparecen al final. No requieren accion. Healthy connections appear last. No action required.
