ReportesReports

Genera reportes detallados sobre el estado del landscape SAP. Exporta a PDF o Excel, y programa generacion automatica en horarios definidos. Generate detailed reports about the SAP landscape state. Export to PDF or Excel, and schedule automatic generation at defined times.

Tipos de ReporteReport Types

★
Resumen EjecutivoExecutive Summary
Vista de alto nivel del estado del landscape SAP. Incluye KPIs principales, tendencias de salud, alertas criticas resueltas/pendientes, y resumen de operaciones del periodo. Ideal para presentaciones a gerencia y stakeholders. High-level view of the SAP landscape state. Includes main KPIs, health trends, resolved/pending critical alerts, and operations summary for the period. Ideal for management and stakeholder presentations.
PDF Excel
✓
Cumplimiento SLASLA Compliance
Reporte detallado de cumplimiento de SLAs por sistema y periodo. Muestra disponibilidad, tiempos de respuesta, ventanas de batch, RTO/RPO, y detalle de cada incumplimiento con causa raiz y acciones correctivas tomadas. Detailed SLA compliance report by system and period. Shows availability, response times, batch windows, RTO/RPO, and detail of each breach with root cause and corrective actions taken.
PDF Excel CSV
⚠
Auditoria de SeguridadSecurity Audit
Revision exhaustiva de seguridad: usuarios con accesos criticos, certificados proximos a expirar, intentos de login fallidos, cambios de autorizacion, y cumplimiento de politicas de seguridad. Incluye hallazgos clasificados por severidad. Comprehensive security review: users with critical access, certificates nearing expiration, failed login attempts, authorization changes, and security policy compliance. Includes findings classified by severity.
PDF Excel
△
Planificacion de CapacidadCapacity Planning
Proyeccion de crecimiento de recursos para los proximos 3, 6 y 12 meses. Incluye analisis de tendencia de CPU, memoria, disco y tablespace HANA. Genera recomendaciones de escalamiento con estimacion de costos y fechas criticas. Resource growth projection for the next 3, 6, and 12 months. Includes CPU, memory, disk, and HANA tablespace trend analysis. Generates scaling recommendations with cost estimates and critical dates.
PDF Excel JSON
★
Revision de IncidentesIncident Review
Analisis post-mortem de incidentes del periodo. Incluye timeline de cada incidente, sistemas afectados, impacto en SLA, causa raiz, acciones de mitigacion, y recomendaciones para prevencion. MTTR y MTTD por categoria. Post-mortem analysis of period incidents. Includes timeline of each incident, affected systems, SLA impact, root cause, mitigation actions, and prevention recommendations. MTTR and MTTD by category.
PDF Excel

Referencia APIAPI Reference

POST /api/reports/generate

Genera un reporte bajo demanda. Generates a report on demand. 

POST /api/reports/generate
{
  "type": "executive_summary",
  "format": "pdf",
  "dateRange": { "from": "2026-03-01", "to": "2026-03-12" },
  "systems": "all"
}

Response: {
  "reportId": "rpt_abc123",
  "status": "generating",
  "estimatedSeconds": 45,
  "downloadUrl": null
}

GET /api/reports/:id

Obtiene el estado y URL de descarga de un reporte. Gets the status and download URL of a report. 

GET /api/reports/rpt_abc123

Response: {
  "reportId": "rpt_abc123",
  "type": "executive_summary",
  "status": "completed",
  "format": "pdf",
  "downloadUrl": "/api/reports/rpt_abc123/download",
  "generatedAt": "2026-03-12T10:05:30Z",
  "expiresAt": "2026-06-10T10:05:30Z"
}

POST /api/reports/schedules

Crea una programacion de reporte automatico. Creates an automatic report schedule. 

POST /api/reports/schedules
{
  "type": "sla_compliance",
  "cron": "0 6 1 * *",
  "format": "pdf",
  "recipients": ["ops-team@company.com"],
  "systems": "production"
}

Response: {
  "scheduleId": "sch_def456",
  "status": "active",
  "nextRun": "2026-04-01T06:00:00Z"
}
