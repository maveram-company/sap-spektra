Reglas de Eliminacion en Cascada Cascade Delete Rules

⚠️
Cuidado: Todas las relaciones usan onDelete: Cascade. Al eliminar una Organization se eliminan TODOS sus datos. Al eliminar un System se eliminan sus componentes, instancias, hosts, metricas, alertas, etc. Caution: All relations use onDelete: Cascade. Deleting an Organization removes ALL its data. Deleting a System removes its components, instances, hosts, metrics, alerts, etc.
