Vista General Overview

La pagina de Landscape proporciona una vista topologica completa del landscape SAP de la organizacion. Muestra todos los sistemas, sus componentes, instancias, hosts y las dependencias entre ellos. Esta vista es crucial para entender la arquitectura del landscape y detectar problemas de configuracion. The Landscape page provides a complete topological view of the organization's SAP landscape. It shows all systems, their components, instances, hosts and dependencies between them. This view is crucial for understanding landscape architecture and detecting configuration issues.

Descubrimiento de Instancias Instance Discovery

Cada sistema SAP consiste en multiples instancias que se ejecutan en uno o mas hosts. El agente descubre automaticamente la topologia de instancias usando sapcontrol y los puertos estandar de SAP: Each SAP system consists of multiple instances running on one or more hosts. Dependencias de Red Network Dependencies

Cada sistema tiene dependencias que pueden ser conexiones a bases de datos, llamadas RFC a otros sistemas, conexiones HTTP o envio de IDocs. El endpoint GET /metrics/systems/:systemId/dependencies retorna las dependencias de un sistema: Each system has dependencies that can be database connections, RFC calls to other systems, HTTP connections or IDoc sending. The GET /metrics/systems/:systemId/dependencies endpoint returns a system's dependencies:
