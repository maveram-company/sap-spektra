Gestion de alertas y escalamiento Alert management and escalation

Flujo de EscalamientoEscalation Flow

En la parte superior de la pagina se muestra una tarjeta con el flujo visual de escalamiento. Este flujo representa como las alertas no atendidas se escalan automaticamente a traves de tres niveles: At the top of the page, a card with a visual escalation flow is displayed. This flow represents how unattended alerts are automatically escalated through three levels:

L1
Advertencia — 30 min Warning — 30 min
→
L2
Peligro — 60 min Danger — 60 min
→
Admin
Primario — 120 min Primary — 120 min

El flujo se lee de izquierda a derecha. Si una alerta no se gestiona dentro del tiempo asignado al nivel actual, se escala automaticamente al siguiente nivel. Los tiempos indicados (30, 60 y 120 minutos) representan el tiempo maximo antes del escalamiento. The flow reads from left to right. If an alert is not handled within the time assigned to the current level, it automatically escalates to the next level. The indicated times (30, 60, and 120 minutes) represent the maximum time before escalation.

Contiene los siguientes elementos: Below the escalation flow, there is a filter toolbar that allows you to control which alerts are displayed on the page. It contains the following elements:

Grupo de Botones de EstadoStatus Button Group

Un grupo de 4 botones que permiten filtrar alertas por su estado actual: A group of 4 buttons that allow filtering alerts by their current status:

ActivasActive TodasAll CriticasCritical ResueltasResolved

Dropdown de SistemaSystem Dropdown

Un menu desplegable que permite filtrar alertas por un sistema SAP especifico. Al seleccionar un sistema, solo se muestran las alertas correspondientes a ese SID. A dropdown menu that allows filtering alerts by a specific SAP system. When you select a system, only alerts corresponding to that SID are shown.

Contador de AlertasAlert Count

Junto a los filtros se muestra un contador que indica cuantas alertas coinciden con los filtros seleccionados actualmente (por ejemplo, "12 alertas"). Next to the filters, a counter shows how many alerts match the currently selected filters (for example, "12 alerts").

Tarjetas de AlertaAlert Cards

Las alertas se muestran como tarjetas individuales apiladas verticalmente. Cada tarjeta de alerta contiene varias secciones de informacion: Alerts are displayed as individual cards stacked vertically. Each alert card contains several information sections:

Badges del EncabezadoHeader Badges

La parte superior de cada tarjeta de alerta muestra una fila de badges informativos: The top of each alert card shows a row of informative badges:

  • Badge de nivel — Indica la severidad de la alerta con un icono y color: Critica (rojo, icono de alerta), Advertencia (ambar, icono de precaucion), Informativa (cian, icono de informacion). Level badge — Indicates alert severity with an icon and color: Critical (red, alert icon), Warning (amber, caution icon), Informational (cyan, info icon).
  • Badge de escalamiento — Muestra el nivel de escalamiento actual: L1 o L2. Escalation badge — Shows the current escalation level: L1 or L2.
  • Badges de estado — Indicadores adicionales como "Activa", "En gestion", "Resuelta", etc. Status badges — Additional indicators such as "Active", "Acknowledged", "Resolved", etc.
Informacion del SistemaSystem Information

Debajo de los badges, cada alerta muestra los datos del sistema afectado: Below the badges, each alert shows the data of the affected system:

  • SID — Identificador del sistema SAP donde se origino la alerta (por ejemplo, S4P). SID — SAP system identifier where the alert originated (for example, S4P).
  • System ID — Identificador interno del sistema en SAP Spektra. System ID — Internal system identifier in SAP Spektra.
  • Hora — Fecha y hora en que se genero la alerta. Time — Date and time when the alert was generated.
Titulo y MensajeTitle and Message

Cada alerta incluye un titulo descriptivo del problema detectado y un mensaje con detalles adicionales sobre la condicion que disparo la alerta. Por ejemplo: Each alert includes a descriptive title of the detected problem and a message with additional details about the condition that triggered the alert. For example:

⚠
CPU Usage Critical
Uso de CPU al 98% en el servidor de aplicacion hanadb01 del sistema S4P. Se requiere accion inmediata. CPU usage at 98% on application server hanadb01 of system S4P. Immediate action required.
Informacion de GestionAcknowledgment Information

Cuando un operador toma una alerta en gestion, la tarjeta muestra informacion de quien se hizo responsable: When an operator acknowledges an alert, the card shows information about who took responsibility:

  • Tomada por — Nombre del operador que acepto la alerta. Acknowledged by — Name of the operator who accepted the alert.
  • Cuando — Fecha y hora en que se tomo la alerta. When — Date and time when the alert was acknowledged.
Informacion de ResolucionResolution Information

Las alertas resueltas muestran una caja verde de resolucion con los siguientes datos: Resolved alerts display a green resolution box with the following data:

✓
  • Categoria — Tipo de resolucion aplicada (por ejemplo, "Falsa alarma", "Resuelto manualmente", "Auto-resuelto"). Category — Type of resolution applied (for example, "False alarm", "Manually resolved", "Auto-resolved").
  • Nota — Comentario del operador explicando como se resolvio el problema. Note — Operator comment explaining how the issue was resolved.
  • Resuelto por — Nombre del usuario que cerro la alerta. Resolved by — Name of the user who closed the alert.
  • Cuando — Fecha y hora de la resolucion. When — Date and time of the resolution.

Modal de ResolucionResolution Modal

Al hacer clic en el boton "Resolver", se abre un dialogo modal que permite registrar la resolucion de la alerta. El modal contiene los siguientes elementos: When clicking the "Resolver" (Resolve) button, a modal dialog opens allowing you to record the alert resolution. The modal contains the following elements:

Componentes del ModalModal Components

Dropdown de CategoriaCategory Dropdown

Un menu desplegable con categorias predefinidas de resolucion. Debes seleccionar una categoria que mejor describa como se resolvio la alerta. Las categorias disponibles incluyen: A dropdown menu with predefined resolution categories. You must select a category that best describes how the alert was resolved. Available categories include:

  • Resuelto manualmente — El operador realizo acciones manuales para corregir el problema. Manually resolved — The operator performed manual actions to fix the problem.
  • Auto-resuelto — El problema se corrigio automaticamente sin intervencion. Auto-resolved — The problem was fixed automatically without intervention.
  • Falsa alarma — La alerta fue generada erroneamente; no habia problema real. False alarm — The alert was generated erroneously; there was no real problem.
  • Runbook ejecutado — Se ejecuto un runbook automatizado que resolvio el problema. Runbook executed — An automated runbook was executed that resolved the problem.
  • Escalado a proveedor — Se abrio un ticket con SAP u otro proveedor. Escalated to vendor — A ticket was opened with SAP or another vendor.
  • Otro — Otra categoria no listada anteriormente. Other — Another category not listed above.

Nota de ResolucionResolution Note

Un campo de texto (textarea) donde puedes escribir una nota libre explicando los detalles de la resolucion. Este campo es opcional pero recomendado para mantener un historial claro de las acciones tomadas. A text field (textarea) where you can write a free-form note explaining the resolution details. This field is optional but recommended to maintain a clear history of actions taken.

Botones del ModalModal Buttons

El modal presenta dos botones en la parte inferior: The modal presents two buttons at the bottom:

  • Cancelar — Cierra el modal sin guardar cambios. La alerta permanece en su estado actual. Cancelar (Cancel) — Closes the modal without saving changes. The alert remains in its current state.
  • Confirmar — Registra la resolucion con la categoria y nota proporcionadas. La alerta pasa al estado "Resuelta" y muestra la caja verde de resolucion en su tarjeta. Confirmar (Confirm) — Records the resolution with the provided category and note. The alert changes to "Resolved" status and displays the green resolution box on its card.
⚠
Importante: Una vez resuelta, la alerta no puede volver a abrirse. Asegurate de que el problema esta efectivamente corregido antes de confirmar la resolucion. Important: Once resolved, the alert cannot be reopened. Make sure the issue is effectively fixed before confirming the resolution.
