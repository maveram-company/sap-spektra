Definiciones de MetricasMetric Definitions

🕒
MTTR

Mean Time To Repair — Tiempo promedio para restaurar un sistema a operacion normal despues de un incidente. Se calcula como el promedio de (resolvedAt - createdAt) para todas las alertas criticas del periodo. Mean Time To Repair — Average time to restore a system to normal operation after an incident. Calculated as the average of (resolvedAt - createdAt) for all critical alerts in the period. 

MTTR = AVG(resolvedAt - createdAt)
// WHERE severity = 'critical'
// AND status = 'resolved'
📈
MTBF

Mean Time Between Failures — Tiempo promedio entre incidentes criticos. Un MTBF alto indica estabilidad del sistema. Se mide en horas. Mean Time Between Failures — Average time between critical incidents. A high MTBF indicates system stability. Measured in hours. 

MTBF = totalUptime / incidentCount

// Period: last 30 days

DisponibilidadAvailability

Availability % — Porcentaje del tiempo que el sistema estuvo operativo (health score >20) durante el periodo. Se excluyen ventanas de mantenimiento planificadas. Availability % — Percentage of time the system was operational (health score >20) during the period. Planned maintenance windows are excluded. 

Availability = (uptime / (total - planned)) * 100
// uptime: minutes with healthScore > 20
// planned: maintenance window minutes

Puntuacion de Salud (Health Score)Health Scoring

El health score (0-100) es un indicador compuesto que refleja el estado general del sistema. Se calcula en tiempo real y se almacena historicamente para analisis de tendencias. The health score (0-100) is a composite indicator reflecting the overall system state. It is calculated in real time and stored historically for trend analysis.

// Health score calculation
const healthScore = Math.round(
  (100 - cpu) * 0.20 +           // CPU factor
  (100 - memory) * 0.20 +        // Memory factor
  (100 - disk) * 0.15 +          // Disk factor
  dbScore * 0.20 +               // Database factor
  alertScore * 0.15 +            // Alert factor
  connectivityScore * 0.10       // Connectivity factor
);

// Clamp to 0-100 range
