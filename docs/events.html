Proposito del Feed de EventosEvent Feed Purpose

El feed de eventos centraliza todos los sucesos relevantes del entorno SAP en un unico flujo cronologico. A diferencia de las alertas (que requieren accion), los eventos son registros informativos que documentan la actividad del sistema, los cambios de configuracion, las acciones de usuario y los incidentes de seguridad. The event feed centralizes all relevant happenings in the SAP environment into a single chronological stream. Unlike alerts (which require action), events are informational records that document system activity, configuration changes, user actions and security incidents.

El feed de eventos se actualiza automaticamente en tiempo real. Los nuevos eventos aparecen en la parte superior de la tabla con una animacion de entrada. Fuentes de EventosEvent Sources

🖥
SAP

Eventos generados directamente por los sistemas SAP monitoreados. Incluyen metricas del sistema, jobs, dumps ABAP, cambios de configuracion y estados de procesos. Events generated directly by monitored SAP systems. Include system metrics, jobs, ABAP dumps, configuration changes and process states.

  • Dumps ABAP y errores de runtimeABAP dumps and runtime errors
  • Finalizacion de jobs (SM37)Job completions (SM37)
  • Alertas de umbrales de metricasMetrics threshold alerts
  • Cambios de parametros del sistemaSystem parameter changes
  • Estados de work processesWork process states
⚙
Platform

Eventos generados por la plataforma SAP Spektra. Incluyen operaciones de la API, eventos de cache, health checks de conectores y actividad administrativa. Events generated by the SAP Spektra platform itself. Include API operations, cache events, connector health checks and administrative activity.

  • Health checks de conectoresConnector health checks
  • Operaciones de cache RedisRedis cache operations
  • Cambios de configuracionConfiguration changes
  • Eventos de deploy y reinicioDeploy and restart events
  • Errores de integracionIntegration errors
🔒
Security

Eventos relacionados con la seguridad de la plataforma. Incluyen intentos de autenticacion, cambios de permisos, actividad sospechosa y renovacion de tokens. Events related to platform security. Include authentication attempts, permission changes, suspicious activity and token renewals.

  • Logins exitosos y fallidosSuccessful and failed logins
  • Cambios de rol y permisosRole and permission changes
  • Renovacion/expiracion de JWTJWT renewal/expiration
  • Intentos de acceso no autorizadoUnauthorized access attempts
  • Bloqueo de cuentasAccount lockouts
