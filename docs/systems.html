Sistemas SAPSAP Systems

Monitoreo completo del landscape SAP — todos sus sistemas en una sola vista. Full SAP landscape monitoring — all your systems in a single view.

Vista general de la paginaPage overview

La pagina Sistemas (Monitor > Sistemas) es el punto central para supervisar la salud y el estado de cada instancia SAP conectada a Spektra. Al ingresar, usted ve un encabezado que muestra el titulo "Landscape SAP" junto con un subtitulo descriptivo — "Vista completa de todos los sistemas monitoreados" — y un indicador del numero total de sistemas registrados. En la esquina superior derecha del encabezado aparece un boton azul-violeta etiquetado "Conectar Sistema" que lo lleva al asistente de incorporacion de nuevos sistemas. The Systems page (Monitor > Systems) is the central hub for overseeing the health and status of every SAP instance connected to Spektra. Upon entering, you see a header displaying the title "Landscape SAP" alongside a descriptive subtitle — "Complete view of all monitored systems" — and a count of the total registered systems. In the upper-right corner of the header a blue-violet button labeled "Conectar Sistema" (Connect System) takes you to the new-system onboarding wizard.

El boton "Conectar Sistema" abre la pantalla de configuracion donde puede vincular un nuevo sistema SAP mediante Cloud Connector, agente Spektra u otro metodo. The count next to the subtitle reflects how many systems are registered in your Spektra tenant. The "Connect System" button opens the configuration screen where you can link a new SAP system via Cloud Connector, Spektra agent, or another method.

8. Breaches

La pestana Breaches lista todas las violaciones de umbrales activas para este sistema especifico. Cada breach muestra: The Breaches tab lists all active threshold violations for this specific system. Each breach shows:

MetricaMetric UmbralThreshold Valor actualCurrent value SeveridadSeverity DesdeSince
CPU Usage ≤ 85% 93% Critico 2026-03-11 08:14
Dialog Response Time ≤ 1000 ms 1450 ms Warning 2026-03-11 09:32

La columna Umbral muestra el limite configurado para la metrica, Valor actual muestra la lectura que disparo la alerta, y Desde indica cuando comenzo la violacion. Las breaches activas alimentan el puntaje de salud del sistema que se muestra en la tarjeta principal. The Threshold column shows the configured limit for the metric, Current value shows the reading that triggered the alert, and Since indicates when the violation started. Active breaches feed into the system health score shown on the main card.

⚠️
Las breaches criticas generan alertas automaticas segun las reglas de notificacion configuradas en Configuracion. Revise la seccion de Alertas para gestionarlas. Critical breaches trigger automatic alerts based on the notification rules configured in Settings. Check the Alerts section to manage them.
