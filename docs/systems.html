Referencia APIAPI Reference

GET /api/systems

Lista todos los sistemas con paginacion y filtros opcionales. Lists all systems with pagination and optional filters. 

GET /api/systems?environment=PRD&status=healthy&page=1&limit=20

Response: {
  "data": [...],
  "meta": { "total": 24, "page": 1, "limit": 20 }
}

GET /api/systems/:sid

Obtiene detalle completo de un sistema incluyendo metricas, componentes y dependencias. Gets full system detail including metrics, components, and dependencies. 

GET /api/systems/S4P

Response: {
  "sid": "S4P",
  "description": "S/4HANA Production",
  "healthScore": 96,
  "type": "S4HANA",
  "database": "HANA 2.0",
  "environment": "PRD",
  "instances": 3,
  "hosts": 2,
  "components": [...]
}
