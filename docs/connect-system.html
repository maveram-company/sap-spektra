Conectar SistemaConnect System

Asistente paso a paso para registrar un nuevo sistema SAP en Spektra Step-by-step wizard to register a new SAP system in Spektra
🔗
Puedes acceder a este asistente desde multiples lugares: el boton "Conectar Sistema" en la pagina de Administracion, desde el Dashboard, o desde la barra lateral haciendo clic en Admin → Conectar Sistema. You can access this wizard from multiple places: the "Connect System" button on the Administration page, from the Dashboard, or from the sidebar by clicking Admin → Connect System.

Haz clic en la tarjeta que corresponda a tu tipo de infraestructura: The first screen of the wizard presents two large cards for you to choose the connection method. Click the card that corresponds to your infrastructure type:

Tras seleccionar un metodo, la tarjeta elegida se resalta con un borde cian y fondo iluminado. Haz clic en "Continuar" para avanzar al primer paso del flujo seleccionado. After selecting a method, the chosen card is highlighted with a cyan border and illuminated background. Click "Continue" to proceed to the first step of the selected flow.

Paso 2: InstalacionStep 2: Installation

Segun el sistema operativo seleccionado en el paso anterior, se genera automaticamente el comando de instalacion correspondiente. El comando se muestra en un bloque de codigo con un boton de copiar en la esquina superior derecha. Based on the operating system selected in the previous step, the corresponding installation command is automatically generated. The command is shown in a code block with a copy button in the top-right corner.

  --token "tok_install_••••••••"

Windows (PowerShell)

📋
irm https://install.spektra.maveram.com/agent.ps1 | `
  iex -OrgId "org_01hx4k7m9p" `

AIX / HP-UX / Solaris

📋
wget -qO- https://install.spektra.maveram.com/agent-unix | \

Nota: Los tokens de instalacion (tok_install_*) se generan automaticamente y son unicos para tu organizacion. Note: Installation tokens (tok_install_*) are automatically generated and unique to your organization. They are masked for security on screen, but clicking the copy button copies the full value to your clipboard.

Paso 4: VerificacionStep 4: Verification

El ultimo paso ejecuta una prueba de conexion entre Spektra y tu sistema SAP. Al llegar a este paso, la verificacion comienza automaticamente y veras un spinner animado que gira mientras se realiza la prueba. The final step runs a connection test between Spektra and your SAP system. Upon reaching this step, the verification starts automatically and you'll see an animated spinner that rotates while the test is in progress.

Verificando conexion... Verifying connection...
Probando conectividad con el agente y sistema SAP Testing connectivity with agent and SAP system
✓
Conexion exitosa Connection successful
El sistema S4P se ha registrado correctamente System S4P has been registered successfully

Si la verificacion falla, se muestra un icono rojo con un mensaje de error descriptivo y sugerencias para resolver el problema (verificar firewall, credenciales, etc.). Puedes volver al paso anterior para corregir los datos y reintentar. If the verification fails, a red icon is shown with a descriptive error message and suggestions for troubleshooting (check firewall, credentials, etc.). You can go back to the previous step to correct the data and retry.

✗
Error de conexion Connection error
No se pudo establecer conexion con el agente. Verifica que el agente esta instalado y el puerto 8443 esta abierto. Could not establish connection with the agent. Verify the agent is installed and port 8443 is open.

Paso 3: VerificacionStep 3: Verification

La verificacion para la ruta del Cloud Connector prueba la conectividad del tunel entre Spektra y el sistema SAP a traves del Cloud Connector. El comportamiento visual es el mismo que en la ruta del agente: The verification for the Cloud Connector path tests the tunnel connectivity between Spektra and the SAP system through the Cloud Connector. The visual behavior is the same as in the agent path:

Un spinner animado gira mientras se verifica la conectividad del tunel. An animated spinner rotates while the tunnel connectivity is being verified.
Si la prueba es exitosa, aparece un icono verde con marca de verificacion y un mensaje de confirmacion. If the test succeeds, a green checkmark icon appears with a confirmation message.
Si falla, aparece un icono rojo con X, un mensaje de error y sugerencias para resolver el problema. If it fails, a red X icon appears with an error message and troubleshooting suggestions.

Navegacion del AsistenteWizard Navigation

En la parte inferior de cada paso del asistente encontraras dos botones de navegacion que te permiten moverte entre los pasos: At the bottom of each wizard step you'll find two navigation buttons that let you move between steps:

← AtrasBack ContinuarContinue →
  • Atras — Regresa al paso anterior sin perder los datos ingresados. En el primer paso, regresa a la seleccion de metodo. Back — Returns to the previous step without losing entered data. On the first step, returns to method selection.
  • Continuar — Avanza al siguiente paso. El boton se desactiva si los campos obligatorios no estan completos. Continue — Advances to the next step. The button is disabled if required fields are not filled.
