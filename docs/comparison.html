Comparacion Comparison

Compara metricas entre sistemas de una misma linea SAP Compare metrics across systems within the same SAP line

Requisito del plan Plan requirement

La funcionalidad de Comparacion requiere que tu plan incluya el modulo comparison. Si no esta incluido, veras un banner de upgrade en lugar del contenido de comparacion. The Comparison feature requires your plan to include the comparison module. If it is not included, you will see an upgrade banner instead of the comparison content.

🔒 Funcionalidad premium Premium feature

Si ves este banner, tu plan no incluye Comparacion de Sistemas. Contacta a tu administrador o actualiza tu plan. If you see this banner, your plan does not include System Comparison. Contact your admin or upgrade your plan.

Selector de linea SID SID line selector

En la parte superior encontraras un dropdown que permite elegir una linea SID. Cada linea agrupa los sistemas SAP que comparten la misma funcion pero operan en distintos entornos (DEV, QAS, PRD). At the top you will find a dropdown to select a SID line. Each line groups SAP systems that share the same function but operate in different environments (DEV, QAS, PRD).

💡
La linea ERP es la unica con multiples entornos (ED1 DEV, EQ1 QAS, EP1 PRD). Las demas lineas (BW, SOL, CRM, GRC, PO, EWM) tienen un solo sistema cada una. The ERP line is the only one with multiple environments (ED1 DEV, EQ1 QAS, EP1 PRD). The remaining lines (BW, SOL, CRM, GRC, PO, EWM) each have a single system.

Tarjetas de sistema por entorno System cards by environment

Para la linea seleccionada, se muestran tarjetas ordenadas de DEV → QAS → PRD. Si la linea tiene 3 sistemas, las tarjetas se distribuyen en 3 columnas. Si tiene 1, se muestra en una sola columna angosta. For the selected line, cards are shown ordered DEV → QAS → PRD. If the line has 3 systems, cards are distributed in 3 columns. If it has 1, a single narrow column is shown.

Elementos de cada tarjeta Card elements

  • Borde superior de color — azul para DEV, amarillo para QAS, violeta para PRD. Identifica visualmente el entorno de un vistazo. Color-coded top border — blue for DEV, yellow for QAS, violet for PRD. Visually identifies the environment at a glance.
  • SID y badge de entorno — la esquina superior izquierda muestra un icono de servidor y el SID (p. ej., ED1). La esquina superior derecha muestra un badge de color con el nombre del entorno. SID and environment badge — the top-left shows a server icon and the SID (e.g., ED1). The top-right shows a colored badge with the environment name.
  • Tipo y descripcion — debajo del encabezado, se muestra el tipo de sistema (p. ej., S/4HANA) seguido de un guion y la descripcion. Type and description — below the header, the system type (e.g., S/4HANA) is shown followed by a dash and the description.
  • Health Gauge — un medidor circular que muestra el puntaje de salud del sistema (0-100). Los colores se adaptan al estado (verde = saludable, amarillo = degradado). Health Gauge — a circular gauge showing the system health score (0-100). Colors adapt to status (green = healthy, yellow = degraded).
  • Status badge — etiqueta con punto indicador y texto del estado (Healthy, Degraded, etc.). Status badge — label with dot indicator and status text (Healthy, Degraded, etc.).
  • Metricas CPU/Memoria/Disco — tres columnas con las metricas principales. Los valores atipicos (outliers) se resaltan en rojo: se marca como outlier el valor mas alto de CPU, Memoria o Disco cuando hay diferencias entre entornos. CPU/Memory/Disk metrics — three columns with main metrics. Outlier values are highlighted in red: the highest CPU, Memory or Disk value is marked as outlier when there are differences across environments.
  • Informacion SAP — seccion inferior con SAP Release, Kernel version, SAP Notes y Cliente. Los valores que difieren entre entornos se resaltan en amarillo, facilitando la deteccion de inconsistencias. SAP information — bottom section with SAP Release, Kernel version, SAP Notes and Client. Values that differ across environments are highlighted in yellow, making it easy to spot inconsistencies.

Grafica de barras comparativa Comparison bar chart

Cuando la linea seleccionada tiene mas de un sistema, se muestra una grafica de barras agrupadas que compara 4 metricas entre todos los entornos: Health Score, CPU (%), Memoria (%) y Disco (%). When the selected line has more than one system, a grouped bar chart is displayed comparing 4 metrics across all environments: Health Score, CPU (%), Memory (%) and Disk (%).

Tabla de consistencia de landscape Landscape consistency table

Cuando la linea tiene multiples entornos y existen datos de validacion disponibles, se muestra una tabla de consistencia de landscape. Esta tabla realiza una validacion cruzada de configuraciones entre los entornos DEV, QAS y PRD. When the line has multiple environments and validation data is available, a landscape consistency table is shown. This table performs cross-validation of configurations across the DEV, QAS and PRD environments.

Lineas con un solo sistema Lines with a single system

Si seleccionas una linea SID que contiene un unico sistema (p. ej., BW, SOL, CRM, GRC, PO, EWM), la grafica de barras y la tabla de consistencia no se muestran. En su lugar, veras un mensaje informativo: If you select a SID line that contains a single system (e.g., BW, SOL, CRM, GRC, PO, EWM), the bar chart and consistency table are not shown. Instead, you will see an informational message:

ℹ️

Esta linea SID solo contiene un sistema (PRD). La comparacion de landscape requiere al menos dos entornos (DEV/QAS/PRD). This SID line only contains one system (PRD). Landscape comparison requires at least two environments (DEV/QAS/PRD).

En este caso, la tarjeta del sistema se muestra en una sola columna angosta, con toda la informacion habitual (gauge, metricas, info SAP), pero sin comparacion ni resaltado de diferencias. In this case, the system card is shown in a single narrow column, with all the usual information (gauge, metrics, SAP info), but without comparison or difference highlighting.
