Tests de Paridad Parity Tests

Los parity tests verifican que los proveedores mock y real devuelven datos con la misma forma (mismos campos, mismos tipos). Cada dominio tiene un archivo *.parity.test.ts. Parity tests verify that mock and real providers return data with the same shape (same fields, same types). Each domain has a *.parity.test.ts file.
