Arquitectura Multi-Modo: SAP Spektra opera en 4 modos — REAL, FALLBACK, MOCK y RESTRICTED — con 12 proveedores de dominio, contratos tipados y ProviderResult<T> como salida canonica. Cada dominio resuelve su proveedor segun el modo activo. Multi-Mode Architecture: SAP Spektra operates in 4 modes — REAL, FALLBACK, MOCK, and RESTRICTED — with 12 domain providers, typed contracts, and ProviderResult<T> as canonical output. Each domain resolves its provider based on the active mode.

Inicio Rapido Quick Start

Clonar e instalarClone and install 
git clone https://github.com/maveram-company/sap-spektra.git
cd sap-spektra
Configurar infraestructuraConfigure infrastructure 
cp apps/api/.env.example apps/api/.env
docker-compose up -d
Instalar y preparar datosInstall and prepare data 
cd apps/api && npm install && npx prisma migrate dev && npm run seed
cd ../../frontend && npm install
Iniciar la aplicacionStart the application 
# Terminal 1: API
cd apps/api && npm run dev

# Terminal 2: Frontend
cd frontend && npm run dev
VerificarVerify
