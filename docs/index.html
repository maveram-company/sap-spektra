Inicio Rapido Quick Start

Clonar el repositorioClone the repository 
git clone https://github.com/maveram-company/sap-spektra.git
cd sap-spektra
Iniciar con DockerStart with Docker 
docker-compose up -d
Instalar dependencias y poblar datosInstall dependencies and seed data 
cd apps/api && npm install && npx prisma migrate dev && npm run seed
cd ../../frontend && npm install
Iniciar la aplicacionStart the application 
# Terminal 1: API
cd apps/api && npm run start:dev

# Terminal 2: Frontend
cd frontend && npm run dev
Acceder al portalAccess the portal

AbreOpen http://localhost:5173 y usaand use admin@acme-corp.com / admin123
