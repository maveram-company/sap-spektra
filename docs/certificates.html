Tipos de CertificadoCertificate Types

🔒
SSL/TLS

Certificados para comunicacion HTTPS segura entre clientes y servidores SAP. Incluye certificados wildcard y SAN.Certificates for secure HTTPS communication between clients and SAP servers. Includes wildcard and SAN certificates.

  • Cifrado en transitoEncryption in transit
  • Autenticacion de servidorServer authentication
  • Fiori Launchpad, WebDispatcher, ICM
👤
SAML

Certificados para Single Sign-On entre proveedores de identidad (IdP) y SAP como proveedor de servicios (SP).Certificates for Single Sign-On between Identity Providers (IdP) and SAP as Service Provider (SP).

  • SSO / Single Sign-On
  • Firma de aserciones SAMLSAML assertion signing
  • Azure AD, Okta, ADFS
🛡️
SNC

Secure Network Communication para proteger la comunicacion RFC/DIAG entre sistemas SAP y SAP GUI.Secure Network Communication to protect RFC/DIAG communication between SAP systems and SAP GUI.

  • Comunicacion RFC seguraSecure RFC communication
  • SAP GUI + SNC
  • SAP Router, SAP Trust Center

Proceso de RenovacionRenewal Process

Identificar certificados por vencerIdentify expiring certificates

Revise el dashboard de certificados y filtre por estado "warning" o "critical".Review the certificate dashboard and filter by "warning" or "critical" status.

Generar CSR (Certificate Signing Request)Generate CSR (Certificate Signing Request)

Desde el detalle del certificado, genere un nuevo CSR con los datos actualizados.From the certificate detail, generate a new CSR with updated data.

Enviar a la Autoridad CertificadoraSubmit to Certificate Authority

Envie el CSR a DigiCert, Let's Encrypt, SAP Trust Center u otro CA.Submit the CSR to DigiCert, Let's Encrypt, SAP Trust Center, or other CA.

Instalar nuevo certificadoInstall new certificate

Cargue el certificado renovado y la cadena completa en el sistema SAP correspondiente.Upload the renewed certificate and full chain to the corresponding SAP system.

Verificar y reiniciar serviciosVerify and restart services

Verifique la cadena de confianza y reinicie los servicios ICM/WebDispatcher si es necesario.Verify the trust chain and restart ICM/WebDispatcher services if needed.
