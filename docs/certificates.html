Seguimiento de ExpiracionExpiration Tracking

Spektra monitorea la fecha de expiracion de cada certificado y genera alertas anticipadas basadas en la cantidad de dias restantes (daysLeft). Los umbrales son configurables por organizacion. Spektra monitors each certificate's expiration date and generates early alerts based on days remaining (daysLeft). Thresholds are configurable per organization.

Dias RestantesDays Left EstadoStatus AlertaAlert AccionAction

Informacion del EmisorIssuer Information

Spektra almacena informacion detallada sobre la Autoridad Certificadora (CA) que emitio cada certificado para facilitar el proceso de renovacion. Spektra stores detailed information about the Certificate Authority (CA) that issued each certificate to facilitate the renewal process. 

interface Certificate {
  id:              string;
  systemId:        string;
  sid:             string;
  name:            string;         // e.g., "ICM HTTPS Server"
  type:            CertType;       // SSL | SAML | SNC
  status:          CertStatus;     // ok | warning | critical
  issuer: {
    commonName:    string;         // e.g., "DigiCert Global Root G2"
    organization:  string;         // e.g., "DigiCert Inc"
    country:       string;         // e.g., "US"
  };
  subject: {
    commonName:    string;         // e.g., "sap-prd.company.com"
    organization:  string;
    san:           string[];       // Subject Alternative Names
  };
  serialNumber:    string;
  algorithm:       string;         // e.g., "SHA256withRSA"
  keySize:         number;         // e.g., 2048, 4096
  validFrom:       ISO8601;
  validTo:         ISO8601;
  daysLeft:        number;         // Calculated field
  fingerprint:     string;         // SHA-256 fingerprint
  sapStore:        string;         // e.g., "STRUST", "SMICM", "PSE"
  autoRenew:       boolean;        // If auto-renewal is configured
  lastRenewed:     ISO8601 | null;
  organizationId:  string;
  createdAt:       ISO8601;
  updatedAt:       ISO8601;
}

Flujo de RenovacionRenewal Workflow

Alerta de ExpiracionExpiration Alert
Spektra genera una alerta warning cuando faltan 90 dias para la expiracion. Se notifica al equipo via email, Slack, y Teams. Se crea un ticket de seguimiento. Spektra generates a warning alert when 90 days remain until expiration. The team is notified via email, Slack, and Teams. A tracking ticket is created.
Solicitud de RenovacionRenewal Request
El operador genera un CSR (Certificate Signing Request) desde la transaccion STRUST del sistema SAP o automaticamente via el agente Spektra. The operator generates a CSR (Certificate Signing Request) from the SAP system's STRUST transaction or automatically via the Spektra agent.
Emision del CertificadoCertificate Issuance
El CSR se envia a la CA (DigiCert, Let's Encrypt, SAP Trust Center). La CA valida y emite el nuevo certificado. The CSR is sent to the CA (DigiCert, Let's Encrypt, SAP Trust Center). The CA validates and issues the new certificate.
InstalacionInstallation
El nuevo certificado se instala en el almacen correspondiente (STRUST, PSE, SMICM). Se reinicia el servicio ICM si es necesario. El runbook "Certificate Renewal" automatiza este proceso. The new certificate is installed in the corresponding store (STRUST, PSE, SMICM). The ICM service is restarted if needed. The "Certificate Renewal" runbook automates this process.
VerificacionVerification
Spektra verifica automaticamente que el nuevo certificado esta activo, el chain es valido, y la comunicacion HTTPS funciona correctamente. La alerta se resuelve automaticamente. Spektra automatically verifies that the new certificate is active, the chain is valid, and HTTPS communication works correctly. The alert is automatically resolved.
