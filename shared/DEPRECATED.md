# DEPRECATED — Shared UI Schema

These files (`ui-schema.json`, `ui-renderer.js`) were part of the **original SSOT renderer** used by the legacy Express mock server and HTML dashboards.

The active frontend is **`frontend/`** (React 19 + Vite 7) with component-based rendering. UI schemas are no longer used.

These files are kept for reference but are **not used or maintained**.
